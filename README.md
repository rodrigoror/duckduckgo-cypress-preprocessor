# Cypress + Cucumber-Js

Is a simple project to be an example of how to use and configure.

## Installation

Is simple, you just need get the npm packages

```bash
npm install
npx cypress open
```

After, you will need to configure cypress.... E2E, next, next...

## Usage

```bash
#run locally
run cypress test
#run with the cypress interface
npx cypress open
```
